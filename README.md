\# 🗑️ Smart Dustbin



Ultrasonic sensor দিয়ে তৈরি Smart Dustbin যা হাত কাছে আসলে automatically খুলে যায়।



\## 🎥 Demo Video

\[YouTube-এ দেখো](https://youtu.be/Zg8oYmcWu28?si=8uFuHhFd6rCBSC6m)



\## 🛠️ Components

\- Arduino

\- Ultrasonic Sensor (HC-SR04)

\- Servo Motor



\## ⚙️ How it works

হাত কাছে আসলে sensor detect করে এবং servo motor ঢাকনা খুলে দেয়।
